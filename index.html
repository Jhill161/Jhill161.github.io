    Projects

    Phrase Hunter Game

    Phrase Hunter is a phrase guessing game made with vanilla JS, HTML, and CSS. This game uses event delegation to listen to all buttons on keyboard, and compares to active phrase, then displays correct guesses. Buttons are disabled after selection and marked if wrong or correct via colors. Phrsases are selected randomly at start of game, and dynamically displayed on the screen 'tiles' via creating 'li's. Lives are lost and heart icons removed for incorrect guesses, and if 5 lives are lost, game is over and screen displays as such. If phrase is guessed, screen displays winning screen. Start Game button will reset keyboard, select new phrase, and start a new game.
